妖，从白小纯进来穿着那件衣服，就一个个隐隐看出了白小纯的性格，虽然如此，可终究还是欣赏。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“昏迷后，醒了就痊愈？”掌门笑了笑，对于白小纯的解释，没有去在意，毕竟每个弟子都有一些自己的秘密，宗门要求的是归属感，不需要那种完全的掌控，不然的话，必定离心离德。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“白小纯，你的奖励几个月前就下去了，从几个月前，你就是我灵溪宗……荣耀弟子！”掌门说出荣耀弟子这四个字时，心底泛起阵阵古怪，这还是他第一次看到活的荣耀弟子。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;他们之前在大殿内也讨论过，彼此都非常为难，因为荣耀弟子的身份，太重要了，在这之前，给的都是战死之人，还从来没过活人，可如今白小纯居然活蹦乱跳的出现在自己面前……<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;这让他们在刚刚知道这个消息时，一个个都愣了一下。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;而且这个身份还不好收回来，葬礼都办了，功劳也有了，这让他们所有人，都愁，所以之前才让白小纯在外面等了半天。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;商议之后，还是没有办法改变，按照门规，只能继续给白小纯这个身份。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“荣耀弟子？”白小纯一愣，他从来没听过这个称呼，站在那里等了半天，眼巴巴的看着掌门等人，现这大殿内的前辈们，一个个都面色古怪，可却没有继续说出别的奖励后，白小纯忍不住了。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“那个……没了？”白小纯问道。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“没了。”掌门挤出一副笑脸。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯立刻急了，正要开口去讲讲道理，说说自己一路多么的辛苦，多么的九死一生时，李青候干咳一声，他最了解白小纯的性子。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“还不快感谢掌门，荣耀弟子这个称号，至今为止，灵溪宗一共就出过十个，你是这千年内的第一个，也是如今唯一的一个。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;身为荣耀弟子，凌驾于内门之上，是灵溪宗至高荣耀，其血脉后人可享用整个宗门的资源，出生就是内门，并且灵溪宗会庇护血脉家族永久！<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;我灵溪宗内有九大家族，都是荣耀家族，这是开创一个辉煌的耀荣身份。”李青候解释道。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯听到这个解释，愁眉苦脸，整个人的气势一下子没了，可怜兮兮的看着李青候，又看了看掌门。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;他不知道自己该说些什么了，这个荣耀弟子的身份，他算是明白了，看似很不错，可实际上是给死人准备的，其赏赐完全是留给后人，可他还活着啊……此刻白小纯悲痛的现，自己居然沦落为当爹的去嫉妒儿女的程度上了。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“谢……掌门……”白小纯哭丧着脸，有气无力的开口。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“从今天开始，你可以称呼老夫为掌门师兄。”掌门郑远东干咳一声，心里要多别扭有多别有，他之前提出拜师，是因为白小纯已牺牲，可如今对方活着回来，他一想自己这么大年纪了，与一个不到二十岁的小子称兄道弟，心底连连叹息。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“啊？”白小纯睁大了眼，他此刻心情起伏太大，进入大殿时满怀期望，而后现实太残酷，心情一下子跌落谷底，可紧接着，又出现了这种变化。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“鉴于你的功劳，再加上你还没有师尊，所以老夫做主，让你拜老夫的师尊为师，所以从此之后，我就是你的师兄了。”掌门心里更别扭了。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯深吸口气，目中刹那露出激动，他觉得宗门对自己太好了，一想到掌门的师尊，那必定是宗门内的太上长老一个级别的，他立刻就兴奋，双眼冒光。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“这从此之后，谁敢欺负我啊，我师父老厉害了，哈哈！！”白小纯心中狂喜，连忙抱拳深深一拜。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“多谢掌门师兄，师兄，咱们师父呢，我这就去给他老人家请安。”白小纯带着期待，美滋滋的说道。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“不急，他老人家坐化前的画像供在后山，我已安排人去准备，稍后会带你过去。”掌门神色古怪，缓缓开口。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“坐化……供在后山……”白小纯整个人如被天雷轰击，呆在那里，满脑子都是这六个字，半晌之后他才反应过来……自己拜的这个师父，已经……故去了。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“我……”白小纯只觉得七窍生烟，脑海嗡鸣更多，他的心再次跌落谷底，欲哭无泪，他甚至都没注意自己是怎么离开的，一路失魂落魄的跟着掌门去了后山，拜了一张画像为师后，茫然的出了种道山，回到了香云山。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;在香云山上，不少看到他的弟子，一个个都连忙拜见，目中露出好奇，甚至还有人好心的带着茫然的白小纯去看了眼他的墓碑。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;望着墓碑，白小纯觉得天都黑了。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“我……我拜了一张画像为师……”白小纯不知自己怎么回到的院子里，呆呆的坐在木屋前，悲愤起来。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;一连数日，他这里都愁眉苦脸，直至半个月后，他才从这打击中恢复过来。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;哀伤叹气的走出屋舍，准备去找张大胖叙叙旧，可刚刚走出，迎面遇到的外门弟子，在看他后，立刻抱拳深深一拜。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“拜见白师叔。”<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯走出几步忽然一顿，转头双眼冒光，一把拉着那位外门弟子。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“你叫我什么？”<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“白师叔啊，您是掌门师弟，弟子……弟子自然要称呼您为白师叔啊。”那外门弟子一愣，赶紧说道。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯松开了手，双眼光芒越来越亮，心脏砰砰跳动，他现自己这个身份，并不是一无是处，辈分大的吓人……<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;他舔了舔嘴唇，忽然大笑起来，笑声回荡，吓的那位外门弟子赶紧后退，不知道白小纯了什么疯。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;白小纯赶紧收起笑声，干咳一下，摆出前辈的模样，向着对方微微点头，不再去找张大胖，而是满怀期待的去了任务处。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;因为……那里人多。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;与此同时，香云山顶，李青候归来后，立刻闭关，他盘膝坐在闭关之处，沉思良久，大袖一甩，神色凝重，开始炼丹。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;“小纯这孩子性格顽劣，需要帮他准备一件保命法宝，可惜我不擅长炼器，只能开一炉九绝丹，去与丹溪宗交换……这种给凝气弟子准备的法宝，一看就是为子侄之辈，丹溪宗定会大开口。”李青候摇头，没有去在意这一点，哪怕九绝丹对他而言，也需付出很大心血才可炼出，可一想到白小纯这一次的九死一生，他便定气凝神，准备开炉。<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;-----------<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;今晚12点！8<br/>　　<br/>　　&nbsp;&nbsp;&nbsp;&nbsp;<br/>　　<br>　　本站重要通知:请使用本站的免费小说APP,无广告、破防盗版、更新快,会员同步书架,请关注微信公众号
